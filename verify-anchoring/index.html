Independent Origin Verification

Verify an anchor proof

Drop an origin ZIP or any file to verify whether its existence was recorded in the Bitcoin blockchain.

Your file never leaves your browser. All verification is performed client-side.

Drop a file here or click to select Origin ZIP or any file

This verifier is independent infrastructure. It makes zero calls to any Umarise, anchoring.app, or itexisted.app server. Hash verification uses the Web Crypto API. Bitcoin anchoring is verified via OpenTimestamps calendar servers.

This page is open source. You can fork it, host it yourself, and verify without any third-party dependency.

How verification works

  1. Your file (or ZIP) is read entirely in your browser — nothing is uploaded.
  2. If it's an origin ZIP, the certificate and artifact are extracted and the SHA-256 hash is recomputed from the original bytes.
  3. The recomputed hash is compared against the hash stated in the certificate.
  4. If a .ots proof is included, it is verified against public Bitcoin calendar servers to confirm the timestamp.
  5. The result shows whether the file's existence was provably recorded at the claimed time.
