V7 is de spijker. 36px, met glow, bovenaan gecentreerd. Het schilderij hangt eraan via een gouden lijn (1px, 16px lang, gradient). Dit is niet decoratief. Dit is structureel. V7 houdt het bewijs op zijn plek.

Anchored: Solid V7, volle glow, solid wire. Het hangt.

Pending: Dashed V7, pulsend, ghost wire. Het wordt opgehangen.

proof.ots bij pending: De dot naast proof.ots pulst mee. Zodra Bitcoin bevestigt: alles wordt solid.

Alles zichtbaar. Niets uitgelegd.

Hash: volledig, één regel, JetBrains Mono 11px, 30% opacity. Wie het kent herkent het. Wie het niet kent ziet een code die bij het plaatje hoort.

Proof components: certificate · hash · proof.ots — als één regel, punten ertussen, 10px, ghost. Dit is wat er in de ZIP zit. Geen file-iconen, geen hints, geen uitleg.

Nul woorden uitleg. Geen "your file stays on your device." Geen "only the hash leaves." Als het interface dat moet zeggen, is het interface fout.

Save. Niet Download (te technisch). Niet Keep (te emotioneel). Save is universeel. Save is wat je doet met iets dat je wilt bewaren. Het is het juiste woord.

States: "Save" → "Saving..." → "✓" → auto-advance Wall (0.8s)
