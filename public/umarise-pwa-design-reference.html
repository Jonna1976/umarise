6. Key Design Rules

  • No pure black. All dark backgrounds use deep dark green (#050A05, #0F1A0F).
  • No onboarding. No tutorials, no "what is" screens. The gesture is known.
  • No navigation menu. V7 hexagon is the only nav element.
  • No scroll on capture. Everything fits one viewport.
  • Minimum font size: 17px body, 13px mono, 11px whispers.
  • Three fonts only: Playfair Display (ceremony), EB Garamond (body), JetBrains Mono (data).
  • The PWA is an action, not a destination. Capture → Sealed → Wall. Done.
  • Files never stored on platform. ZIP stays on device. Only the hash/proof leaves.
  • Auto-advance: After "✓ Owned" → 0.8s → auto-transition to Wall.
  • Share Sheet flow: "Upload and share" in detail modal prompts user to select existing ZIP → shares via native Share Sheet with verify link.
