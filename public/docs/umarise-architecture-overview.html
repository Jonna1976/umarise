Path A — Partner API

Path B — itexisted.app

INSERT → pages

Both paths converge
origin_attestations
origin_id · hash · hash_algo · captured_at · short_token · api_key_prefix

3 · Bitcoin Anchoring

OTS Worker

Hetzner VPS · Hourly cron · Node.js
1. Read unstamped hashes from origin_attestations
2. Aggregate into Merkle tree
3. Submit batch to 3 OTS Calendar Servers
4. Calendar anchors Merkle root in Bitcoin transaction
Block confirmed
Bitcoin Blockchain
Block height · Merkle root · Immutable timestamp
Public. Immutable. Anyone can verify independently.
.ots proof stored
core_ots_proofs

5 · Proof Layers
L1
Anchored Existence

Anchored Signature
L1 + WebAuthn device-bound signature.
Included with every anchor

Anchored Identity
L2 + verified identity binding via attestant.
EUR 4.95 per attestation
L4
Anchored QES
L3 + Qualified Electronic Signature via QTSP.
On request · Requires QTSP integration

8 · Critical Invariant

No column for content exists in any table.

This is architectural, not policy. There is no mechanism to store file content even if someone tried.
Deze bytes bestonden no later than Bitcoin block T.
