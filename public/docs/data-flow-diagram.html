Data Flow Diagram

Umarise Core API — Two paths, one truth. Content never leaves the device.
Version 1.0 17 February 2026 Status: For External Reviewers

Overview

Two paths produce the same result. Content never leaves the device. Only SHA-256 hashes cross any network boundary.

Critical Invariant

No column for content exists in any table. This is architectural, not policy. There is no mechanism to store file content even if someone tried. The hash is a one-way function: content cannot be derived from it.
