5 · Proof Layers
L1
Anchored Existence
SHA-256 hash anchored via OpenTimestamps to Bitcoin. Proof that these exact bytes existed no later than a specific block.
No cost · No account required
L2
Anchored Signature
L1 + WebAuthn device-bound signature. The anchor is cryptographically tied to a specific hardware key (Face ID, fingerprint, device PIN).
No cost · Included with every anchor
L3
Anchored Identity
L2 + verified identity binding. A certified independent attestant confirms the link between signer and anchor via KYC or notary verification.
EUR 4.95 per attestation
L4
Anchored QES
L3 + Qualified Electronic Signature via QTSP. ETSI-conform format (XAdES/PAdES). Legal equivalence to a handwritten signature under eIDAS.
On request · Requires QTSP integration
