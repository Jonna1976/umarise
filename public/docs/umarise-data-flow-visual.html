5 · Proof Layers
L1
Anchored Existence
SHA-256 hash anchored via OpenTimestamps to Bitcoin. These exact bytes existed no later than block T.
Free · No account required
L2
Anchored Signature
L1 + WebAuthn device-bound signature. Anchor cryptographically tied to a specific hardware key.
Included with every anchor
L3
Anchored Identity
L2 + verified identity binding. Independent attestant confirms signer–anchor link via KYC or notary.
EUR 4.95 per attestation
L4
Anchored QES
L3 + Qualified Electronic Signature via QTSP. ETSI-conform (XAdES/PAdES). eIDAS legal equivalence.
On request · Requires QTSP integration
